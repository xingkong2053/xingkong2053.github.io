Intelij idea创建Java Web工程并测试Servlet

创建并设置Java web工程

一些信息

  • idea版本：2019.2.1
  • jdk版本：1.8.0
  • tomcat版本：9.0.24

步骤

  1. 创建web工程

    1.1 File–>New–>Project

    1.2 勾选Web Application

    1.3 填写名称

  2. Web工程设置

    2.1 在WEB-INF目录添加classes lib 两个文件夹

    2.2 File–>Project Structure，进入Project Structure窗口

    2.3 选择modules–>paths选项卡，勾选Use module compile output path，并将Output path和Test output path改为之前创建的classes目录

    2.4 选择modules–>Dependencies，点击右侧+号，选择JARs or derectories，选择创建的lib目录

    2.5 配置打包方式：点击Artifacts选项卡，勾选Include in project build和Show content of elements其他保持默认

  3. tomcat配置

    3.1 在窗口的右上角点击Add Configration...，点击+号，选择tomcat–>local，添加tomcat容器

    3.2 选择Deployment 选项卡，选择+号，选择Artifact...，并修改Application context

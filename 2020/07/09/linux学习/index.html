发表于 2020-07-09 阅读次数：

linux

实用指令

基本命令

reboot 重启

pwd 显示当前路径

ls 显示目录或文件

​ -a 显示所有文件

​ -l 以列表的方式显示信息

cd ~ 回到自己的家目录

uname -a查看系统核心信息 p622

用户指令

su root 普通用户切换到root用户

ll 显示当前文件夹下的所有文件

useradd [username] 创建一个新的用户

useradd -d [file path] [username] 创建一个新的用户并指定家目录

passwd [username] 为用户指定密码

userdel [username] 删除用户保留家目录 ，删除用户需root

userdel -r [username] 删除用户并删除家目录

id [username] 查询用户

su - [username] 切换用户

exit 回退到之前的用户

whomi 查看当前用户

groupadd [groupname] 创建用户组

groupdel [groupname] 删除用户组

useradd -g [groupname] [username] 在组内创建用户

usermod -g [groupname] [username] 将用户修改到组

其他指令

init 012346 指定运行级别

man [option] 帮助

cp source dest 拷贝文件到指定的目录

​ -r 递归复制整个文件夹

\cp 强制覆盖已经存在的文件

mv 移动或这重命名文件（路径相同）

cat 浏览文件（以只读的方式）

运行级别

运行级别说明

0 ：关机

1：单用户（找回丢失密码）

2：多用状态没有网络服务

3：多用户状态有网络服务

4：系统未使用保留给用户

5：图形界面

6：系统重启

etc/initlab 系统运行级别配置文件

相关文件

/etc/passwd 用户配置文件（用户信息）

/etc/group 组配置文件（组信息）

/etc/shadow 口令配置文件（密码和登录信息，加密）

tar -zxvf解压文件tar.gz

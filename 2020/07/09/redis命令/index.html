redis命令

Last Update:2019-12-20 Reading Time: 1 min Readed:

描述 命令
从docker中使用redis数据库 docker exec -it redis /bin/bash
进入redis命令行 # redis-cli
退出redis命令行 exit
添加键值对 set k1 hello
根据键获取值 get k1
向一个字符串中追加 append k1 world
获得全部配置信息 config get *
更改配置信息（不用重启） config set maxmemory 2mb

redis持久化：

rdb:redis database

aof:append only file
